Step 1: Download a Code Editor

Create a new file with the name of your choice.

I will create a file named download_file.py

Step 2: Start Coding

The first step in the coding part is to import requests:

                    import requests

Then, declare the variable url and set it equal to the URL you want to download the file from. 

                    url = "https://file-examples.com/storage/fe9d743740654a8139a48e1/2017/02/zip_10MB.zip"

Next, declare the variable webpage and set it to requests.get(url):

                    webpage = requests.get(url)

Finally, declare the variable futureName, which will take the last element of your list:

                    futureName = url.split('/')[-1]

Step 3: Save and Run

Now that we have declared all the variables, we can write the actual file by taking the content of the response from the web server and writing it to a local file, as follows: 

                    import requests
    
url = "https://file-examples.com/storage/fe9d743740654a8139a48e1/2017/02/zip_10MB.zip"

webpage = requests.get(url)
    
futureName = url.split('/')[-1]
    
with open(futureName, 'wb') as file:
    file.write(webpage.content)

Save the code and run it! To run it, open a terminal, and write python followed by your file name, as in this example: 

                    python download_file.py

Whatever URL you input, it will download the file from it.
